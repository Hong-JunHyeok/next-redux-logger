# ✔️ next-redux-logger

# Usage

```sh
npm install --save next-redux-logger
```

or

```sh
yarn add next-redux-logger
```

## 🟨 JavaScript

```javascript
import NextReduxLogger from "next-redux-logger";
```

```js
const store = createStore(modules, applyMiddleware(NextReduxLogger));
```

## 🟦 TypeScript

```ts
import NextReduxLogger from "next-redux-logger";
```

```ts
const store = createStore(modules, applyMiddleware(NextReduxLogger));
```
